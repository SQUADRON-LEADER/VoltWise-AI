# 📖 About The Project

VoltWise AI is an AI-powered energy forecasting platform developed to accurately predict the **Net Electrical Energy Output (PE)** generated by Combined Cycle Power Plants (CCPP).

Power generation systems operate under continuously changing environmental conditions. Small variations in temperature, pressure, humidity, and vacuum conditions can significantly impact electrical energy production.

Traditional mathematical approaches often struggle to efficiently capture the complex nonlinear relationships between these variables.

VoltWise AI addresses this challenge by leveraging **Artificial Neural Networks (ANN)** to learn hidden patterns from operational data and generate highly accurate predictions.

The platform transforms raw environmental measurements into actionable intelligence through an intuitive and interactive dashboard.

This project combines multiple domains, including:

* 🤖 Artificial Intelligence
* 🧠 Deep Learning
* 📊 Data Analytics
* ⚡ Energy Forecasting
* 🖥 Interactive Dashboard Development
* ☁ Cloud Deployment

The ultimate objective is to improve operational efficiency and support intelligent decision-making in modern energy systems.

---

# 🚨 Problem Statement

Combined Cycle Power Plants are highly dependent on environmental conditions.

Several parameters directly influence energy generation efficiency, including:

* Ambient Temperature (AT)
* Exhaust Vacuum (V)
* Atmospheric Pressure (AP)
* Relative Humidity (RH)

These variables exhibit complex nonlinear relationships that are difficult to model using conventional approaches.

VoltWise AI solves this challenge by implementing an Artificial Neural Network capable of learning these relationships and accurately forecasting electrical energy output in real time.

# 🧠 Machine Learning Model

VoltWise AI utilizes an Artificial Neural Network designed specifically for regression tasks.

The model learns hidden nonlinear relationships between environmental variables and electrical energy generation.

## Model Architecture

```text
Input Layer      → 4 Neurons

Hidden Layer 1   → 6 Neurons (ReLU)

Hidden Layer 2   → 6 Neurons (ReLU)

Output Layer     → 1 Neuron

Task             → Regression
```

## Clone Repository

```bash

cd VoltWise-AI
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

```bash
python3 -m venv venv

source venv/bin/activate
```

